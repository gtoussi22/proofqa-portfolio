---

##  Objectif de ce repo

Ce repository documente l'intégralité du travail QA réalisé sur **ProofQA** — 
une plateforme Managed SaaS de Quality Assurance pour les startups tech africaines.

Il démontre ma capacité à couvrir le cycle QA complet, de la stratégie jusqu'aux 
livrables certifiés, en tant que QA Engineer et Test Lead.

## Structure du repository
proofqa-qa-lab/
│
